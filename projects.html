My Projects

VISNAV - Visor Integrated System for Navigation

Details: VISNAV is a comprehensive motorcycle safety system designed to improve rider awareness and safety on the road. By integrating IoT, augmented reality, and GPS technology, it provides real-time alerts to riders about the proximity of heavy vehicles, enhancing their safety during travel. The system also includes crash detection and navigation features, optimizing comfort for long rides. The project uses ESP32, YOLO v10 for object detection, and Flask for backend development.

CAKES - Chromosomal Analysis Kit for Efficient Screening

Details: CAKES is a machine learning-based medical tool developed to assist in efficient chromosomal screening. The project integrates the YOLO algorithm for image analysis, achieving a high 0.96 confidence level in identifying chromosomal abnormalities. The system processes cytogenetic images to detect genetic disorders with accuracy, enhancing the overall diagnostic process. Built using Flask and Python, it’s optimized for use in medical labs for fast and reliable genetic screening.

SHOPZEE

Details: SHOPZEE is a Python-based web scraping tool designed to collect and analyze product reviews from multiple e-commerce platforms. It utilizes BeautifulSoup4 to extract detailed information and performs sentiment analysis on reviews to provide insights into consumer opinions. The system helps users make informed purchase decisions by highlighting trends and sentiments across various product listings. Additionally, it includes a customizable dashboard that allows users to filter and analyze the data efficiently.

Zoom Clone

Details: The Zoom Clone project is a real-time video conferencing application built entirely with Python. This application replicates essential features of Zoom, including video and audio calls, screen sharing, and real-time chat. Using WebRTC for peer-to-peer communication and Flask for backend services, the application ensures secure and seamless video interactions. This project showcases my ability to develop scalable applications and integrate real-time communication systems.

Jarvis Clone

Details: The Jarvis Clone is an AI-powered voice assistant created using Python and speech recognition technologies. Designed to respond to user queries and perform tasks like setting reminders, fetching weather updates, and controlling smart devices, this project demonstrates my proficiency in Natural Language Processing (NLP). It uses speech-to-text and text-to-speech conversion to offer a seamless user experience. Built as a personal assistant, it enhances user convenience through automation.
